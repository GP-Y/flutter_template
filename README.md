- 💥 `flutter`最新版本的空安全
- 🍀 `view` 和 `逻辑` 完全解耦
- ⚡ `view` 和 `state` 自动响应
- 📦  `dio`、`shared_preferences`等通用模块的封装
- 🔥 去`context`化
- 🎨 `theme`、`color`、`font`、`dimen`等资源文件的统一管理
